# MeloList

A catalog for the music you've lived with.

MeloList is a music tracking and cataloging app — think Letterboxd for albums, or AniList for music. Users build a library of the artists, albums, and tracks they've engaged with, rate and review them, track physical collections, and see their taste emerge on a stats page. No streaming, no scrobbling, no recommendations. You tell it what you've listened to, it holds up a mirror.

## Stack

- **Monorepo**: Turborepo
- **Web**: Next.js 16, Tailwind 4
- **API**: Bun + Elysia
- **Worker**: Bun
- **Database**: PostgreSQL + Drizzle
- **Auth**: Better Auth
- **Hosting**: Railway

Status: early development. No public API yet, but the schema is shaping up. The goal is to have a working MVP by the end of April.
